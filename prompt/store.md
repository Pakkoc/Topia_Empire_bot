색상 선택권에 대해서 변경이 있을 예정이다. 이를 듣고 어떻게 진행할 것인지
말하라.

### 일반 색상선택권

/상점 -> 색상변경권을 개수로 살수있게 한다. (색상을 선택하는 것이 아님)
-> /인벤토리 -> 몇개를 보유하고 있냐에 따라서 -> 구매가능한 색상코드가 나온다.
-> 해당 색상코드를 선택하면은 해당 색상코드와 연결된 역할을 부여한다.
-> 단, 일반 색상선택권 일회성이기 때문에 만약 다른 색상을 선택을 한다면 기존
색상(즉, 기존 색상에 대한 역할)은 제거 -> 새로운 색상에 대한 역할만 보유한다.

### 기간제 색상선택권

/상점 -> 무제한 색상변경권(30일)을 개수로 살수있게 한다.
-> /인벤토리 -> 사용가능한 색상코드가 나온다. -> 해당 색상코드를 선택하면은
해당 색상코드와 연결된 역할을 부여한다.
-> 단, 기간제 색상선택권은 만약 다른 색상을 선택을 한다면 기존 색상(즉, 기존
색상에 대한 역할)은 제거 -> 새로운 색상에 대한 역할만 보유한다.

- 일반 색상선택권 vs 기간제 색상선택권
  일반 색상선택권은 색상을 변경할때마다 "일반 색상선택권이 차감이 되는 방식"
  기간제 색상선택권은 해당 기간내에 내가 원하는 색상을 무제한으로 바꿀 수 있다.

- 기간제 색상선택권의 기간이 만료가 되었을때
  현재 적용중인 색상에 대한 역할을 제거한다.

### 일반 (종류)선택권

/상점 -> (종류)선택권을 개수로 살수있게 한다. (역할을 직접 선택하는 것이 아님)
-> /인벤토리 -> 몇개(종류)선택권를 보유하고 있냐에 따라서 -> 구매가능한 역할목록이 나온다. -> 목록중 원하는 역할을 선택하면은 연결된 역할을 부여한다.
-> 단, 일반 (종류)선택권 일회성이기 때문에 만약 다른 역할을 선택을 한다면 기존 역할(즉, 보유하고 있던 역할)은 제거 -> 새로이 선택한 역할만 보유한다. << 해당 내용 (이전역할 제거는 on,off 기능이 필요할 것 같다

### 기간제 (종류)선택권 - 프리미엄 라인으로 출시 예정

/상점 -> 무제한 (종류)선택권(일수)을 개수로 살수있게 한다.
-> /인벤토리 -> 선택가능한 역할 목록이 나온다. -> 목록중 원하는 역할을 선택하면은 연결된 역할을 부여한다.
-> 단, 기간제 (종류)선택권은 만약 다른 역할을 선택을 한다면 기존 역할(즉, 보유하고 있던 역할)은 제거 -> 새로이 선택한 역할만 보유한다. << 해당 내용 (이전역할 제거는 on,off 기능이 필요할 것 같다)

- 일반 (종류)선택권 vs 기간제 색상선택권
  일반 색상선택권은 역할을 변경 또는 추가구매 할때마다 "일반 (종류)선택권이 차감이 되는 방식"
  기간제 (종류)선택권은 해당 기간내에 내가 원하는 역할을 무제한으로 바꿀 수 있다.

- 일반 (종류)선택권 의 경우 소모품이며 기간에 상관 없이 계속 구매한 역할에 대해 이전 역할 회수를 따로 설정하지 않는 이상 지속적 소유가 가능하다

- 기간제 (종류)선택권 기간이 만료가 되었을때
  현재 적용중인 역할을 회수한다.

상점등록(웹) : 명칭(ㅇㅇ선택권)
소모되는 (인식되는) 아이템 : 상점에 등록된 물품 목록 중 선택
소모개수 (가격) : 2 ~ 자유설정 가능
교환가능한 역할 추가 > 자유롭게 역할추가 가능

상점등록 : 명칭(ㅇㅇ선택권) 기간제
소모되는 (인식되는) 아이템 : 상점에 등록된 물품 목록 중 선택
소모개수 (가격) : 0 (기간제의 경우 소모되지 않을테니 0으로
교환가능한 역할 추가 > 자유롭게 역할추가 가능
