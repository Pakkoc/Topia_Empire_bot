루비/토피 상점(shop)에서 우리는 다른 사람도 이용할 때 디폴트 값을 제공해줄 예정이다.
세부 사항(가격, 역할 등)은 사용자가 직접 조정해야한다.

도커로 DB가 켜져있으니 이를 확인하고 하고 마이그레이션을 진행해야한다.

# 인벤토리 형과 역할 지급에 대한 설명

인벤토리형, 역할 지급 모두 상점에서 구매할 수 있다.

- 인벤토리 형은 인벤토리에있으면 백에서 알아서 해주기때문에 수치 조정만 커스텀 가능하다.
- 역할 지급형은 특정 역할이 있으면 그에 해당하는 기능이 백에서 작동하게 하거나 역할을 통한 xp규칙, xp설정을 통해서 추가적인 설정이 가능하다.

- 이 모든 것은 커스터마이징 가능하지만 우리가 백에서 작동하는 것들에 대해서는 기본 디폴트값으로 다른 서버도 이용할 수 있게 한다.

### 생각하는 기본 로직

1. 인벤토리형
   세금을 걷는다 -> 개인 인벤토리 DB에서 세금감면권이 있는지 파악한다 -> 있으면 세금을 걷을 때 백엔드에서 세금 감면 로직을 적용(지정한 %) / 없으면 기본 % 적용

## 인벤토리형 에 디폴트 값으로 들어갈 항목

- 색상변경권(일반): 1회 원하는 색상으로 변경하기 (이미 설정해놨음)
- 경고차감권: 티켓 문의를 통해서 경고차감권 사용한다고 이야기한다. --> 밴치마킹 : 하루봇 경고시스템
  - 경고차감권 불가 항목이 있음
  - 사용방법: 티켓 문의를 통해서 경고차감권 사용한다고 이야기
  - 운영진이 항목을 본 후 이걸 확인하고 운영진이 차감해줄지 판단하고, 운영진이 경고차감권을 받아들인다고하면 특정 명령어로 해당 인원의 경고차감권을 인벤토리에서 차감시킨다. (차감할 개수도 선택가능하다.)
- 역할선택권 (이건 타섭전용?) < 일단 만들어야 함 / 색상선택권의 대체품으로 타섭에서 사용될 내용
  - 역할을 수집하는 사람이 특정 돈을 모으면 구매할 수 있게 한다. 이는 다른 역할이 있다고 삭제되지 않고 뺐다 꼈다 할 수 있으면 좋을 것 같다.
- 세금감면권
  - 세금징수시 세금감면권이 있다면 자동으로 즉시 사용으로 인벤토리에서 소모되는 구조
- 이체 수수료 감면권

  - /이체를 했을 때 정보를 모두 입력하고 엔터를 누르면 인벤토리에 세금감면권이 1개 이상 있으면 소모를 할 것인지를 물어본다. 소모를 한다고하면 인벤토리에서 세금감면권 1개가 차감되고 세금이 감면된다. (이 세금은 웹에서 커스텀가능)

- 게임 수수료 감면권 (추후)
  - 게임 시작 전 감면권이 있음 사용을 할것인가 여부 체크 하여 인벤토리 소모형
- 이벤트참여권 (추후)
- 랜덤박스 (추후)

## 역할지급형에 디폴트 값으로 들어갈 항목

- 색상변경권(기간제, 프리미엄): 해당 기간내에 원하는 색상으로 무제한 변경 가능 (이미 설정해놨음)
- 활동부스트권 : 해당 역할이 들어가있으면 가중치 적용 (기간제)
- 프리미엄잠수방 : 프리미엄잠수방 역할이 있으면 특정 채널이 열림 -> 해당 채널가서 잠수를 타면 -> 그냥 잠수타는것보다 더 많은 xp, 토피 받음(이 부분은 역할만 주어주면 xp, 토피설정에서 역할에 대한 배율 설정하면 됨.)
- 기간제 vip 라운지 입장권: 역할이 있으면 특정 채널이 열림
- 기간제 디토 실버&골드 : 실버/골드 역할을 가지고 있으면 금고가 있는데 -> 금고에 넣으면 실버/골드 ---> 디토뱅크 피아 확인

이에 대해서 이해했는지 우선 나한테 말하라.
