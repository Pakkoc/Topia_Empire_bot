## 1️⃣ 화폐 구조 개요

### ■ 토피 (Topy)

* 무상 화폐(=활동형 화폐)
* 서버 내 **모든 기본 경제 활동의 중심**
* 인플레이션 제어를 위해 **세금·수수료 적용 대상**

### ■ 루비 (Ruby)

* 유상 화폐 / 현금 연동 화폐
* 서버 국고 수익의 핵심
* **장터 고액 거래, 환급 대상, 관리자·전속 인력 보상**에 사용
* 세금·수수료 **면제**
