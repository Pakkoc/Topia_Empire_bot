# <a:DH_MINA_09:1462814893546934418> Forge XP 설정

> ## <a:DH_MINA_04:1462813052138291282> 개요 및 목적
> ```- 채팅/음성 활동 기반 XP(경험치) 획득 규칙 설정
> - 텍스트 XP와 음성 XP를 각각 독립적으로 관리
> - 레벨별 필요 XP 커스터마이징 지원```

---

> ## 1단계: XP 획득 설정
> ### <a:DH_MINA_03:1462813029577130200> 텍스트 XP 설정
> * **textXpEnabled**: 텍스트 XP 획득 활성화 여부
> * **textXpMin / textXpMax**: 메시지당 획득 XP 범위 (최소~최대)
> * **textCooldownSeconds**: 다음 XP 획득까지 대기 시간 (초)
> * **textMaxPerCooldown**: 쿨다운 기간 내 최대 획득 횟수
> ```기본값:
> - 최소 XP: 15
> - 최대 XP: 25
> - 쿨다운: 60초
> - 쿨다운당 횟수: 1회```

> ### <a:DH_MINA_03:1462813029577130200> 음성 XP 설정
> * **voiceXpEnabled**: 음성 XP 획득 활성화 여부
> * **voiceXpMin / voiceXpMax**: 지급당 획득 XP 범위 (최소~최대)
> * **voiceCooldownSeconds**: 다음 XP 획득까지 대기 시간 (초)
> * **voiceMaxPerCooldown**: 쿨다운 기간 내 최대 획득 횟수
> ```기본값:
> - 최소 XP: 10
> - 최대 XP: 20
> - 쿨다운: 60초
> - 쿨다운당 횟수: 1회```

---

> ## 2단계: 레벨 설정
> ### <a:DH_MINA_03:1462813029577130200> 기본 공식
> ```레벨² × 100
> 예: 레벨 5 = 2,500 XP```
> <a:DH_MINA_03:1462813029577130200> 커스텀 설정이 없으면 기본 공식이 자동 적용됨

> ### <a:DH_MINA_03:1462813029577130200> 커스텀 레벨 설정
> * **텍스트 레벨**: 텍스트 XP 기준 레벨 테이블
> * **음성 레벨**: 음성 XP 기준 레벨 테이블
> <a:DH_MINA_03:1462813029577130200> 레벨별 필요 XP를 개별적으로 지정 가능

> ### <a:DH_MINA_03:1462813029577130200> 레벨 관리 기능
> * **레벨 추가**: 새로운 레벨 요구량 추가
> * **기본값 복원**: 1~10 레벨을 기본 공식으로 초기화
> * **전체 삭제**: 모든 커스텀 설정 제거 (기본 공식 사용)
> <a:DH_MINA_03:1462813029577130200> 레벨 번호 수정 가능 (1~999)

---

> ## 3단계: XP 타입 분리
> ### <a:DH_MINA_03:1462813029577130200> 텍스트 레벨 vs 음성 레벨
> * 텍스트 XP와 음성 XP는 각각 별도의 레벨 시스템으로 운영
> * 유저는 두 가지 레벨을 동시에 보유
> <a:DH_MINA_03:1462813029577130200> 레벨 보상도 XP 타입별로 독립 설정 가능

> ### <a:DH_MINA_03:1462813029577130200> XP 비활성화 시 동작
> * 텍스트 XP 비활성화 → 텍스트 레벨 탭 비활성화
> * 음성 XP 비활성화 → 음성 레벨 탭 비활성화
> <a:DH_MINA_03:1462813029577130200> 활성화된 XP 타입으로 자동 전환

---

> ## 4단계: 봇 동작 (자동 처리)
> ### <a:DH_MINA_03:1462813029577130200> 텍스트 XP 처리
> * 유저가 메시지 전송 시 자동으로 XP 계산
> * 쿨다운 및 차단 규칙 적용 후 XP 지급
> * 레벨업 시 역할 보상 및 알림 자동 처리
> ```처리 흐름:
> 1. 메시지 감지 → 차단 규칙 확인
> 2. 쿨다운 확인 → XP 범위 내 랜덤 지급
> 3. 레벨업 확인 → 역할 부여/채널 해금
> 4. 알림 채널에 레벨업 메시지 전송```

> ### <a:DH_MINA_03:1462813029577130200> 음성 XP 처리
> * 음성 채널 참여 중 주기적으로 XP 지급
> * 쿨다운 간격마다 XP 자동 계산
> * 혼자 있는 채널에서는 XP 미지급
> ```처리 흐름:
> 1. 음성 채널 참여 감지
> 2. 쿨다운 주기마다 XP 지급 시도
> 3. 레벨업 시 역할 부여/채널 해금
> 4. 알림 채널에 레벨업 메시지 전송```

> ### <a:DH_MINA_03:1462813029577130200> 레벨 동기화
> * 레벨 요구량 변경 시 전체 유저 레벨 재계산
> * 역할 보상 변경 시 자동으로 역할 재적용
> <a:DH_MINA_03:1462813029577130200> 대시보드에서 변경 시 봇이 자동 동기화
