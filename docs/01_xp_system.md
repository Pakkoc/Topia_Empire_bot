# XP 레벨링 시스템

## 개요

디스코드 서버의 XP 및 레벨링 시스템 기능 명세

---

## 기본 설정

- 기본 세팅은 **프로봇 기준치**로 초기 설정
- 이후 직접 수동 설정 가능

---

## XP 획득 설정

### 텍스트 XP
| 항목 | 설정값 |
|------|--------|
| 획득량 | N XP |
| 쿨타임 | N분마다 N회 획득 |

### 음성 XP
| 항목 | 설정값 |
|------|--------|
| 획득량 | N XP |
| 쿨타임 | N분마다 N회 획득 |

---

## 핫타임 설정

특정 시간대에 XP 획득량 증가 설정 가능

### 텍스트 XP 핫타임
- N XP, N분마다 N회 획득

### 음성 XP 핫타임
- N XP, N분마다 N회 획득

---

## 레벨업 시스템

- 레벨당 **N XP** 채워야 레벨업 가능
- XP 기준치 달성 시 자동 레벨업

---

## 보상 시스템

- XP 기준치 달성 시 보상 지급
- 이전 역할 제거 여부 설정 가능

---

## 제외 설정

### XP 제외 채널
- XP를 받지 못하는 채널 설정 가능

### XP 제외 역할
- 경험치를 받지 못할 역할 설정
- 예: 미인증 유저 등

---

## 알림 설정

- 레벨업 시 알림 발송 채널 설정

---

## 채널 해금

- 특정 레벨 달성 시 해금되는 채널 설정 가능
